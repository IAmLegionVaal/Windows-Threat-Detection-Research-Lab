# Release Readiness

## Completed

- Six experimental Windows detection rules
- ATT&CK and data-source mapping
- Positive and negative synthetic fixtures
- Automated YAML and metadata validation
- Automated expected-match and clean-negative regression tests
- Analyst triage guidance
- CI artifact packaging

## Release classification

The pack is suitable as a **portfolio and lab release candidate**. Rules remain experimental and require SIEM-specific conversion, local field mapping, allow lists, and production tuning before operational deployment.

## Merge gate

The pull request may be merged once the latest CI run passes and the experimental status remains clearly documented.
